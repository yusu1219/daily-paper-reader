## 摘要
在生物网络中识别对动态具有显著影响的节点是系统生物学中的核心问题，尤其对于基因调控网络中干预靶点的优先排序尤为重要。本文提出了一种基于Shapley值的框架，用于评估布尔网络中节点相对于给定目标节点的重要性。该框架包括两种互补的度量：Knock-out Shapley值和Knock-in Shapley值。此外，我们提出了一种基于传播的方法，实现了这些Shapley值的高效计算。该方法通过利用网络的逻辑结构，避免了穷举模拟。对于无环网络，该方法能够得到精确结果；对于有环网络，也能提供良好的近似。在Cell Collective数据库的基准模型上进行的评估表明，该传播方法在显著提升计算速度的同时，能够准确恢复节点重要性排序。

## Abstract
Identifying dynamically influential nodes in biological networks is a central problem in systems biology, particularly for prioritizing intervention targets in gene regulatory networks. In this paper, we propose a Shapley-value-based framework for assessing the importance of nodes in a Boolean network with respect to a given target node. The framework comprises two complementary measures: the Knock-out and the Knock-in Shapley values. Moreover, we present a propagation-based method that enables their efficient computation. By exploiting the logical structure of the network, the method avoids exhaustive simulations. The approach is exact for acyclic networks and provides good approximations for cyclic networks. Evaluation on benchmark models from the Cell Collective database shows that the propagation method accurately recovers node importance rankings while achieving substantial speed-ups.

## 二、方法论：Shapley 框架与传播算法

### 1. 核心思想
- **Shapley值引入**：将网络节点视作“博弈玩家”，每个节点的贡献通过其对目标节点（Target）的激活状态影响来定义。Knock-out (KO) 与 Knock-in (KI) 对应节点被固定为 0 或 1 的两种实验情况。
- **关键思想**：通过计算每个节点在所有输入组合下对目标节点状态的平均边际贡献，得到其重要性。  
- **两种指标：**
  - **Knock-out Shapley 值**：衡量将节点永久关闭（值为 0）后的影响。
  - **Knock-in Shapley 值**：衡量将节点永久激活（值为 1）后的影响。

### 2. 传播式计算方法（Propagation Method）
- **问题**：直接模拟全部输入组合需复杂度 \(O((n+m)^2\times 2^n)\)，计算代价过高。
- **解决方案**：
  - 将一般布尔网络转换为**二值化布尔网络（Binarized Boolean Network, BBN）**，保证每个节点最多只有两个输入。
  - 将计算“是否节点状态改变会影响目标节点”的问题等价为真值表行集传播：
    1. 从目标节点出发，向上层传播“哪些输入组合导致目标变化”的行集；
    2. 根据逻辑运算符（AND/OR/NOT）的规则筛选相关输入；
    3. 对于结构复杂的“菱形”与“循环”结构：
       - 菱形结构通过局部模拟修正；
       - 循环结构采用迭代传播直至收敛（或达到阈值）。
  - 在**无环网络**中可得到完全精确结果；在**有环网络**中为近似。
- **复杂度改进**：
  - 将总体复杂度从 \(O((n-m)2^n)\) 降为 \(O((n+m)\times 2^n)\)。
  - 对稀疏网络近似线性提升。

### 3. 正确性证明
- 作者在附录中提供形式化证明：对无环网络，其Propagation算法计算结果严格等价于标准Knock-out/Knock-in仿真。

## 三、实验设计

### 1. 数据集与实验场景
- **主要数据来源**：  
  - 来自 **Cell Collective 数据库**的 20 个基准模型；
  - 其中 18 个含反馈循环、2 个为无环模型；
  - 每个模型选取 2–15 个输入节点；
  - 为每个模型随机或依据功能选取 3 个目标节点。
- **代表案例**：
  - Fibroblast信号通路模型（139 节点）；
  - T 细胞受体信号模型（94 节点）。

### 2. 对比基线方法
- **标准仿真法**：通过完整的 knock-out/knock-in 扫描模拟计算真实 Shapley 值。
- 在延伸对比中（Sec. IV-D）还与以下方法进行定性比较：
  - Determinative Power (DP)
  - Strength 指标
  - Dynamical Impact 方法（基于邻接矩阵特征向量）

### 3. 评价指标
- **主指标：**
  - 相对均方误差（relative RMSE）
  - 排名性能指标：Normalized Discounted Cumulative Gain (NDCG)  
- **辅助指标：**
  - Kendall’s τb、Spearman’s ρ，用于比较节点排名一致性；
  - 正确传播行比例（验证传播精度）。

- 传播算法在多数模型上可**准确恢复节点影响力排序**：

  - 方法能识别出已知关键调控节点；
